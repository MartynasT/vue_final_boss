Class .ml-(1-5) adds margin-left;

.p-2

.p-4

Display

Block:

Class .block sets display to block;

None / hidden:

Class .d-none or .hidden sets display to none;

Flex:

Class .flex sets display to flex;

Flex align/justify:

Class .justify-between - justify-content: space-between;

Class .justify-around - justify-content: space-around;

Class .justify-center - justify-content: center;

Class .center - justify-content: center and align-items: center;

Class .flex-center - display:flex, justify-content: center and align-items: center;

Class .flex-wrap - flex-wrap: wrap;

Class .align-center - align-items: center;
